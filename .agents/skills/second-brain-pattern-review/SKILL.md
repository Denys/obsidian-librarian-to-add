# Second Brain Pattern Review Skill

Use this workflow after the Second Brain reference material has been inspected and summarized.

## Purpose

Review generated Obsidian notes for practical knowledge-management quality.

## Checks

1. The note is useful for future retrieval.
2. The note is small enough to reuse.
3. The source connection is visible.
4. Links are meaningful.
5. The note remains in a reviewable staging flow.

## Current limitation

The detailed source material is not yet present in the repository.

## Output

Return findings, suggested improvements, missing references, and proposed eval cases.
