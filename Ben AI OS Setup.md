# 5 Skills to Build a Second Brain

Five Claude Code skills that take you from an empty folder to a working second brain, then run it on autopilot. Set it up, operate it, optimize it, share it with your team, and connect it to Claude from anywhere.

## Skill 1. OS Setup 🚀

[os-setup.zip](Ben%20AI%20OS%20Setup/os-setup.zip)

**/os-setup** — Spin up your full vault structure and personalize it in 2 questions. Empty folder to working second brain in 5 minutes.

## Skill 2. OS Operator ⚙️

[os-operator.zip](Ben%20AI%20OS%20Setup/os-operator.zip)

**/os-operator** — Builds a personalized operator prompt and schedules it to run your second brain on autopilot.

## Skill 3. OS Optimizer ✨

[os-optimizer.zip](Ben%20AI%20OS%20Setup/os-optimizer.zip)

**/os-optimizer** — 9-framework audit of your vault. Every finding ships with a fix. You approve, it applies.

## Skill 4. Team OS 🤝

[team-os.zip](Ben%20AI%20OS%20Setup/team-os.zip)

**/team-os** — Turns your single-user vault into a shared team workspace. Adds permissions so different team members see different parts of the vault. Built on the open-source Relay sync layer.

## Skill 5. OS MCP 📡

[os-mcp.zip](Ben%20AI%20OS%20Setup/os-mcp.zip)

**/os-mcp** — Connects your vault to Claude from anywhere. After this runs, you can ask Claude to read or update your vault from Claude Desktop, the Claude apps, or any device. Deploys to Railway, which costs about $5 a month.
